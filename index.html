How It Works

1

Download & Install

Grab the latest release for your platform from GitHub and install the app.

2

Configure Hooks

Use the built-in hook installer or run the setup script to connect Claude Code to the app.

3

Add Your API Keys

Optionally add ElevenLabs and Twilio credentials in settings for voice and SMS features.

4

Start Coding

Launch Claude Code as usual. Sessions appear automatically in the popover as they start.
